一个游戏盲的游戏攻略：巫师联盟

Jul 30, 2019

生物钟乱套了，夜里睡不着更个博客吧，写写玩儿了一个月的巫师联盟。

众所周知我平时生活还是很佛系的，主要体现在于不追剧和不玩游戏。我深知自己的自制力为零，追过的电视剧基本上都是不吃不睡不上班连看七十个小时看下来的，所以除非万不得已，游戏和电视剧我都是不碰的。

不过作为某伤疤眼镜男快二十年的迷妹，六月底听说出了Wizard Unite，我还是火速去下载了一个玩儿。其实下载的时候并不觉得这个游戏能在我手机上呆很久，因为去年玩了一把坑爹的Harry Potter:Hogwarts Mystery没给我无聊的……总之我下载了不到24小时就怒而删除了。

到目前为止，巫师联盟我已经玩了一个多月了，昨天刚升到21级。这一个月以来热度确实慢慢减退了，从一开始坐地铁走路吃饭都盯着手机，还会专门抽时间坐在哪个花园里打要塞，现在则差不多只有实在无聊才会想起来看一眼。但至少我一个月以来真的每天还会至少上线签到一次，而且工作日基本上也能在坐地铁的时候把任务做完，说明这个游戏的黏性还是不错的。

游戏玩法

这是官网的预告片……

游戏是PokemonGo的公司做的，大概就是把PokemonGo+早期Pottermore+开心农场(?)放在了一起的AR游戏。

背景故事设定在哈利波特系列中的“19年后”，此时哈利赫敏等人已然在魔法部担任要员。这时候有个之前是奥罗，现在是神秘事务司的缄默人Grim Fawley为了爱情和魔法部反目成仇，为了向麻瓜暴露魔法的存在，他用魔法让麻瓜世界充满魔法痕迹的foundable（我也很想吐槽这个名字……），每个foundable还被一个法力很强的confoundable守护着（再次吐槽名字……）。游戏玩家的任务就是，在生活中找到各种魔法痕迹，找到后大战confoundable，打赢后把其守护的confoundable还到原本的地方。

举个例子：foundable是某斯莱特林同学，被看不见的confoundable变成了一只白鼬（这个梗很赞）。玩家走在路上发现了“魔法痕迹”，点进去以后施咒（也就是在手机屏幕上划两下……）把白釉变回斯莱特林同学。

这个游戏有非常多不同的模块，需要玩上好一会儿才能搞清各模块的相关性。直接帮助你升级的三大模块，上面预告片里面也都有拍到——

  1. 街上的魔法痕迹，看到一个就点进去，战胜confoundable后就能加分了。每施一个咒语耗费一点能量
  2. 进入要塞和狼人/精灵等物种决斗。这个需要你本人在某个要塞附近（比如我公司附近最近的要塞在Bryant Park我就得本人去了那儿才能打）。我虽然没试过，但要塞其实可以和好友一起打，效率会高很多。
  3. 门钥匙。基本上你在街上走着走着就可以捡到门钥匙盒，从你打开门钥匙盒子的那一刻算起你走了2km后，就可以拿到盒子里的门钥匙。你通过门钥匙进入海格小屋，禁林，博金博克或者奥利凡德魔杖店，可以在里面捡分升级。

辅助升级的模块有——

  1. 旅店。和要塞一样是由系统分配的，比如附近几个地铁站就被做成了游戏里的旅店。旅店两个用途，一个是可以进来补充能量，还有就是可以在这里放黑魔法探测器（黑魔法探测器的用途是可以让周围出现更多分值高的foundables）
  2. 温室。也是系统分配地点的。两个用途，一个是补充植物储备，具体补充哪些植物由系统随机分配。还有就是可以在里面种植物，种子和水都可以在街上捡，种完了过一定时间回来收，就有点像开心农场。
  3. 配魔药。原料就是你捡的/种的植物储备和你能在街上捡到的其他原料，要不然从街上捡，要不然从温室里随机抽取。配好的魔药根据种类有不同用途，最常用的有能增加你咒语效力的，还有决斗的时候给自己回血的。
  4. 上课。基本上就是你在做任务/决斗等等过程中，除了金币还有一些其他外挂以外，还会被奖赏羊皮纸。攒够一定羊皮纸就可以去“上课”，也就是花掉几卷羊皮纸，换取你在要塞里决斗的一些技能。


上面这个图，大红顶是要塞，蓝色插红旗的是温室，冒烟的是旅店，太阳形状那个是魔法痕迹，旁边有个小瓶子一样的东西就是魔药原料。
除此之外当然也有各种任务可以做，这些就是后话了。

游戏体验

总结：瑕不掩瑜。

瑕疵非常非常明显，我觉得都列不完。很多小伙伴们开始没玩多久就被这些尿点给吓退了。

  1. 动不动就闪退。我平均五分钟得重启一次游戏，亏的我真的有耐心……
  2. 定位/定速不准。如果我把游戏关上，走得远一点再打开，它一定会提示我“你移动的太快了，你真的不是在开车吗？”我真的只是没有时时刻刻在盯着手机而已啊。
  3. 加好友功能很奇怪。首先呢加好友要用一串数字，而不是用ID，非常麻烦。其次，我发现加了好友以后，好友完全看不到我的头像，昵称和魔杖信息。问题是设这些本来就是给其他人看的呀。
  4. 就算不算各种闪退，Bug也非常非常多，多到我不知道就这种代码怎么能上线……一开始有很多种魔药和黑暗探测器基本不起作用，更新了一次以后这些问题改了，但还是有其他问题。比如说，有一个成就是“如果你在两个国家都还过foundables就送你某个勋章“。于是我真的兴致勃勃的在冰岛还了几个foundables,结果发现什么也没送。一查，发现我还得主动去改iPhone设置的语言和地域信息😂反之，如果我一直没有出国，光靠改手机设置，其实也是可以拿到这个勋章的……

以上都是一些硬伤，其实更劝退的还是游戏本身的复杂性。我刚上手的时候，除了在街上找黑魔法痕迹然后施咒以外，真的完全不会用其他功能。黑魔法探测器怎么用？门钥匙怎么用？上课又是在干什么？什么叫foundable什么叫confoundable…一开始玩起来真的感觉一头雾水。

不过熟悉起规则以后就越来越觉得很好玩了，而第六级应该说是我体验上的一个转折点——升到第六级，就可以自己配魔药了。

于是我终于知道之前在街上捡的那一大堆乱七八糟的东西都是拿来干嘛的了，而且自己配了药，对敌的时候自己喝，很有种自给自足的成就感呢。

那么这个游戏具体好玩在哪里呢？我觉得最重要的是，游戏设定与罗琳笔下和华纳拍出来的哈利波特/神奇动物世界完全吻合，毫无OOC，让我们这些骨灰级哈迷非常快慰了。

  1. 真的用的是电影的背景音乐。有些我能听出来其实是稍微有修改的，但整体感觉是对的！
  2. 梗玩的非常666。比如有一个情景，foundable是少年纳威，confoundable是斯内普，游戏玩家要施“滑稽滑稽”咒把斯内普身上的衣物换成纳威他奶奶的……这还不是最好笑的，最令人绝倒的当之无愧是亚瑟·韦斯莱先生，具体场景我就不剧透了……
  3. 里面的人和动物都很可爱。几位人类主角和电影中的长相造型基本一模一样，而动物们真的都萌到爆表！


    这个小鹰头马身有翼兽真的好萌好萌啊，萌化我了
  4. 这可能是因为我游戏玩的太少，孤陋寡闻才会有这种感觉——在虚拟世界中碰到三次元小伙伴总觉得特别喜感。比如有一次我进公司附近一个饭店，本来就是想补点能量，结果居然看到那家饭店里放了个黑魔法探测器写的是——我同事的名字。我马上找到同事确认后并加了好友，然后自己在里面又给他补了俩探测器。根据游戏介绍，三个探测器一起用效果是最强的——在十分钟之内，我第一次见到了，少年哈利波特本人的foundable…
