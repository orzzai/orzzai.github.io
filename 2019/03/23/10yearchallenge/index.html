一只过期萝莉的碎碎念

首战托福记

Mar 23, 2019

这是一波迟到了两个月的 #10YearChallenge. 昨天是2019年3月22日。十年前的昨天，我在地铁上过了一遍口语然后再华强北吃了个麦当劳早餐后，怀着紧张雀跃的心情，第一次走进了托福考场。

现在想想，这很可能是我这辈子最难忘的一天。

成绩单上没有名字只有姓。等我考完出来被考官问到此事，我一开始有些一头雾水不知道怎么回事，直到考官和我暗示这个成绩可能不算数。我出了考试那座楼，就在街上开始大哭——也是我懂事之后绝无仅有的体验了。

当时的心情还记得很清楚，两个原因让我特别悲伤。一个是，那之前的两个月，我把自己如此往死里嗑还能坚持下来，完全是为了这一天，而这一天的成绩作废等于是我之前那个月一切努力都付之东流。还有一个是本以为考完了，不管好坏，毕竟之后不用复习了，一想到如果要重考的话等于要把过去那简直不是人过的一个月再经历一次，我就觉得生不如死。

后来确实重考了两次，成绩也都中规中矩，虽说不够满意（都没超过102）但也不算太差。再后来申请学校的时候，这份成绩单也确实正常能用。于是回想起那一天心情经历过的起起伏伏，竟开始有了“站着说话不腰疼”的轻松感——就这么点破事儿激动什么啊，长大以后比这更糟糕更严重更无可挽回的事，多的多呢。

但是十年以来，我从未像考之前那个月一样努力过，即使是找工作找到哭的时候都没有。我想之后也很难再有了。

——我克服了我的弱点（不是缺点，是弱点，不能改的那种），挑战了我的极限，并且在提高效率方面有了吓到自己的发现。回顾一下这段经验，即使是以后在工作中也是会非常宝贵的吧。

记得那会儿是知道申请美国不错的大学，托福100算是个门槛，而和我基础差不多的H童鞋刚好考了101. 她说她每天大概复习3个小时，这让平时拖拖拉拉成绩还不怎么样的我非常犯愁。

我当时的班主任是年级里数一数二的狠角色，作业多要求高，达不到她的要求被她叫到办公室骂一顿的话，能达到literally男默女泪的效果，自尊心被击得灰飞烟灭（我泪过，嗯）。我成绩在班里不上不下地吊中间，每天晚自习结束之前作业总留个尾巴，等着回宿舍打手电筒写完。最怕的就是周五，因为老师们似乎以为一天只要不上课就有无穷多个小时。我总是每周五抱着六七份试卷回家，怎么做也做不完，一般都是周日晚自习甚至周一早上交之前才草草收尾。

总而言之，像我这么一个拖拉的人，我打死都不相信我每天能挤出3个小时备考托福。于是我开始零零散散地想办法凑碎片时间——

  1. 早上6:40起床7:30才正式上课，如果把刷牙漱口吃早餐的时间压缩在20分钟内的话，我可以花半个小时刷一套口语
  2. 下午4:30放学，晚上7:00开始晚自习，一般情况下我会花一个多小时磨磨蹭蹭地洗澡洗衣服吃饭，但非常时期，我应该用这时间刷两篇托福阅读
  3. 晚上10点下自习，10:40睡觉，于是我又多出来半个小时的时间做一整套听力
  4. 课间休息时间戴耳机听“美国之声”

这样就凑够两个小时了，然后一般我还会在课间+英语课刷掉一套阅读（我英语水平应付高中作业/考试还是绰绰有余的），这样加起来也差不多就3个小时了。

而那一个月来我作业从来没做完过。首先我课间和午休都没放过，只要老师哪堂课留作业了基本上一下课就会开始做。之前中午午休时间会回宿舍睡觉，那一个月也免了，每天中午都在教室吭哧吭哧地赶作业。

这也就罢了，最神奇的是我居然练出了每周五睡觉前就把整个周末的作业全部做完的本事，从而周六周日能整套整套地刷模题。这还不是只靠利用碎片时间能达到的——我每天作业做完后，或者哪个课间老师还没留当天的作业，我会根据规律猜老师会留什么作业（一般也就是习题册往后做上一两个章节，或者书上哪几个题目，基本上摸清老师们对题目的喜好可以猜个八九不离十），并且提前把它们给写完。这样就可以留出周六周日两个整天不用写作业啦。

这种神一般的效率我坚持到了3.22，之后当然是垮下来了。我考完一周后二刷，当然对托福的热情已经远远不如第一次考了。二刷完后做值日一个不小心有什么东西忘了收，害得班里被扣分，于是我连着做了两周值日。虽说值日的工作量远远不如托福备考（每天半小时都不到）但我以这个为借口安慰自己的懒惰和拖延，自此我就再也回不到托福备考那个月的状态了。

好了，总结陈词，想要提高工作效率，可以

  1. 努力收集碎片时间，神马lunch break等饭菜的时间，上下班通勤的时间，甚至蹲坑思考人生的时间，都是可以利用起来的！
  2. 预知接下来的工作，如果客户/老师/老板派活儿之前你就已经开始做了，那效果一般是会很惊艳的
