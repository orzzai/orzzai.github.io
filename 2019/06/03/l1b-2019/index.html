Anxiety Management 101 - L1b Extension

Jun 3, 2019

终于终于终于和USCIS的角力告一段落了，加上抽中了H1b，现在的感觉是终于不用卖身为奴看金主脸色过日子了。不过这几个月等L1b extension的日子过的是相当痛苦，写篇文章纪念一下。

开始申请阶段

2018.11.19

“我婚礼定好日期了！！明年3月10”——某青

于是早早在伴娘团预定了席位的我和H童鞋马上就买了机票回国了。我因为知道回国还要办签证，于是给自己留了三周在国内，买了2.22-3.14的往返机票，刚好我有另一波朋友3.7-3.17在纽约，我还可以赶回来和她们一起玩。

2018.11 感恩节后

很久之前律师跟我说公司还在跟移民局续blanket L1 extension，我感恩节回公司以后就发邮件问了问情况。

一天后收到回信——“Blanket没音儿了，你赶快办个individual petition吧。”

这两个有什么区别呢？Blanket petition是公司一办办一堆，一旦办下来了，我就不用给移民局交材料了。

Individual则不同，律师和HR要根据每个人的情况准备材料，然后送到移民局审批……也就是说，如果移民局审批速度不够快的话，我不一定赶得上2.22的飞机。事实证明最后确实没赶上。

2018.12

月初，律师最后问了我一次我当时的年薪和公司地址，我也给她了。后来不知道在忙什么，整个月都没有和她再联系。现在想来挺后悔的，那会儿我怎么就忘了多催催她呢。

2019.1月初

1月2日。新年一过完，我就给律所发了邮件问进度，结果……材料竟然还没寄出去。“你老板昨儿刚在你材料上签好名，现在在公司HR那儿，等他们也签好名了再给我，我再整合好寄给移民局。”

我一看时间赶不及了，赶快给她说我要上premium processing，并且麻溜儿从波多黎各寄了张支票去费城的律所。

一周后律师收到支票，发现……我在支票上没写日期😂

我去年买了个表……

于是我紧急联系菜师傅，叫他从Virginia给我往费城寄了一张支票。

两天后律师收到支票。1月11日，律师终于整理好我全部资料，寄到了德州的移民局。

RFE大作文阶段

2019.1.25

在移民局收到我资料的第11天，我被RFE了。

知道这件事的当时我整个人懵了，基本上陷入一种完全不知道干什么的麻木状态。可能过了有半个小时吧，我终于把情绪收拾清楚，然后给自己列了个需要做的事情的清单——

  1. 给律师发邮件，让她给我一个大概可能的时间表
  2. 微信之前同样被RFE的同事，问问他之前用了多久（他很快就回我了，说至少一个月。我……）
  3. 微信我老板，让他有个心理准备（老板：哦，好吧，以前没处理过这个）
  4. 微信某青，跟她说我多半要缺席了……（这应该是我最怕的一步）
  5. 把H同学约出来买个醉

那晚上在喝了四五杯忘了什么酒，然后和H大半夜进了地铁站还在发疯以后，我的情绪竟然欢乐了起来，并且维持了整个周末。当我决定为自己的情绪调节能力点个赞的时候，我并没有想到，这只是噩梦的开始……

2019.2

1月28日。律所给我发了个邮件，大概让我提供我工作中用到的各种材料。总共分八大类，每大类里面有分四五个小类，每一类我都得提供至少两份材料还有对材料的语言描述。看到这个东西我感觉特别头大，感觉就和上学时候要写paper一样，看到就头痛，只想努力往死线拖拖拖。但我也知道在这件事上只能靠我自己自觉，毕竟我参不参加婚礼和我的老板和律师一点关系都没有，真的去不了的话整个公司也就只有我一个人会伤心。

硬着头皮花了半天把材料都整理出来发了过去，结果老板跟我说“你这些training materials的深度不够啊，你需要证明我公司再从外面招人，招不到一个你这样的。”于是我一咬牙，花了一两个小时，直接把整个公司的文档库打了个包下载下来，当自己的training materials用了。

1月30日，我们终于把这一大堆资料发给了律师。然后律师回复我们，想问我们要产品源代码，还有和客户的合同之类的。

于是我老板和公司HR又为这事儿折腾了一周，当然这个过程中我是没办法掺和的，只能着急的干等。一周后的2月6号，我公司这边终于把相关信息整理好提供给了律所。

2月8日。律师又发邮件来了，问我要我在纽约的组和北京的组所有组员的……job description和简历？不是，我哪里能搞到所有人的简历啊……？纽约的组还好说（只有8个人）我在北京那个组60多个人呢，而且我那会儿的顶头上司老早就离职了好嘛……

关键是律师的邮件里还写的不清不楚的，有个地方还写错了，搞得我和我老板都以为只用提供纽约的组员信息就行。

2月12日。律所又跑来问我要我在美国和国内的工资条和近三年的税单（之前为什么没来找我要啊！？）同时律所也把起草好的RFE response letter发给了我和我老板，让我们查漏补缺，里面空出来了不少地方让我们填，我们大概花了两三天也都补好了。

2月22日。

还有！跟我们说，我在国内的组的组员信息也要我提供一份……

应该说这些任务都不算很难，但是折腾了一个月的我，到了这个节骨眼儿上，已经完全无法忍受接着写这个RFE大作文的过程了。每次看到律师找我要不管什么东西，我都觉得，头痛，头痛，真的好头痛……感觉和上学的时候熬夜做一个完全不会做的作业真的一模一样。

都到了这个时候，我是彻底不可能如期回国了。于是我和律师确认了一下，发现只要公司愿意放人，我也可以在petition被批准之前回国，然后在国内等着petition被批准再去签证。知道这个消息的我算是终于看到了一线希望，觉得我终于可以回去参加婚礼了，于是把机票改到了3.8日到4月底（之前提到过还有一帮小伙伴3.7-3.17来纽约，最后只有3.7晚上见到了两个）。那会儿我并没有想到，之后的两个月，我真的一直在担心会不会丢饭碗……

也就是这段时间，我心情的起伏特别严重，感觉情绪的波动已经完全超出了自己能够控制的范围。基本上，只要收到律所发邮件给我要东西，我就会有想哭想喝醉的冲动；但只要我把什么东西给律所寄了回去，我就会有种，终于……如释重负的轻松感，就想找个地方暴饮暴食。

2月26日

终于把北京组员的名单和学历也都写好上交了（北京这么多人我其实真的不知道谁是什么学历的，一个个查linkedin之类的真的好累啊……）这时候律师又一球打回来——你们写的job description每个人只有一句话啊，太简单了，能不能给个详细点的？

我这会儿都已经疯了——这特么什么时候才算完啊！好在老板是经验丰富的老司机，直接给了我个公司HR的人名，让我找这个人要一份招聘广告，截取了里面的job description就可以发给律师了。

结果公司招聘部门的人也都不是好打发的。老板让我找的那个人大半天没理我，下午跟我说“你去找另一个人”。我找的“另一个人”又跑来问我“你要这个干什么？发给移民局？那你叫律师来找我。”

总之这事儿又绕了好几圈，我们在2月28日才把job description整理出来发了过去。

2019.3月初

3月1日，最终的RFE回复信终于准备完成。由我在纽约的老板 和 在北京的老板的老板（之前提到过我顶头上司离职了所以让他的老板代劳了）都签过名以后，再发给公司HR部门，HR将这些内容在公司信纸上打印出来后，还要将打印件寄给费城的律所。折腾来折腾去，终于，在3月8日，我的RFE response和我本人一起从美国东岸出发，它去德州我回国。

签证阶段

2019.3.20

我的petition终于被批准了！联系好一个3月底从纽约回国的同事帮我把I-797和I-129S带回来以后，我定了4.4在广州领事馆的面签。

2019.4.3

没想到啊没想到，第八次签美国签证的我，竟然在阴沟里翻船了——

是的，签证预约系统里还留着我2017年填进去的DS160号吗，而我并没有把它改成最新的。于是就在这天，我收到领事馆的邮件，叫我4月4号不用去了，等着重新约吧……

这一天过的非常长。我可以接受自己蠢自己粗心，但我非常非常害怕要和我老板报备这件事情……要知道petition没批准就跑回国这么大胆的事直接先斩后奏，如果真的丢了饭碗的话我一点都不奇怪。

2019.4.18

重新预约的签证是这一天，于是下午去广州面签了。过程还是有些bug，比如说我带过去的照片被领事馆说不合格，让我现场重新拍。还有就是面签完以后让我交$120的签证费，可我并不知道要交这笔钱，只好离开领馆找我妈要了钱，再跑回领馆来拿代表“签证通过”的黄条。不过不管怎么说，至少面签到最后是通过了的。

我以为这就快结束了，结果这个尾巴拖的比我想象中的长……

2019.4.29

签证状态在经过10天的administrative processing以后终于变成了issued. 两天后收到护照，签证有效期给到了2020年年底。

这10天以来我天天刷一亩三分地，还发现不少“面签通过结果最后被check“的例子的，每看到一个这样的例子我的心里就往下坠一下……

这个时候我已经花了$250把我回程机票改到了5月6号。万幸万幸，如果再晚几天出签的话，我又要再花这么多钱改机票，这不就变成无底洞了么……

后续

2019.5.6。在飞机上看了16小时《逃避可耻但有用》后，我终于成功到达了美国。下了飞机直接被人拉到入境队伍最前面，于是连队都没排。

然而入境官员填我I-94的时候竟然把我身份写成了L1-A（好像我们公司另外几个和我同期办L1b的都经历过这个）……于是我两天后又专门跑了一趟JFK机场的Deferred Inspection Office改了一次I-94😂

Anyways，也算过了一把高管的瘾？

总结

这个5月过有多忙碌而充实，这之前的几个月就过得有多焦虑而空虚。

我无心学习无心工作，甚至无心好好玩耍。每天除了弹钢琴解压以外，我觉得我基本上就是在慌乱无措——找到办法——等待其他人的配合——在等待期间感到无比焦虑的恶性循环当中度过度过。

一开始，觉得有一个很有效的方法就是好好分解自己的焦虑——当您能想通自己为什么焦虑的时候，你慢慢也就不焦虑了。

到后来觉得这个方法不完全游泳——我知道我焦虑是因为自己不能控制的未知……可是我再努力摆正心态，我还是不能控制这个未知啊！而且只是摆正心态又能怎么样？根本问题不解决，心态再好也只能治标不治本啊。

所以说，这个问题依然是无解的吧。

无所谓啦。至少这一切都已经过去了。在我遇到下一次必须和USCIS斗智斗力的情况时，先好好珍惜我这一年想去哪就能去哪的自由吧。
